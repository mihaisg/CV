Windows 7 Fix

Aplicație folosită pentru a întreține sistemul de operare Windows 7.

Codul și conceptul aplicației au fost realizate în urmastudiului amănunțit al sistemului de operare Windows.

Aplicația a fost scrisă în C# (Visual Studio 2012/NET Framework 3.5) și îndeplinește următoarele funcții:

  • Detecția unui administrator pentru a bloca accesul persoanelor neautorizate evitând astfel defectarea sistemului de operare prin dezactivarea unor servicii sau a unor aplicații importante
  • Aflarea informațiilor despre sistemul de calcul utilizat
  • Eliberarea spațiului de pe hard disk prin ștergerea fișierelor inutile (fișierele temporare, fișiere cache etc)
  • Afișarea erorilor apărute în timpul folosirii sistemului de operare
  • Imbunătățirea performanțelor sistemului de operare prin dezactivarea unor servicii nefolositoare
  • Activarea sau dezactivarea unor funcții ale sistemului de operare (ca de exemplu hibernarea sau funcția Trim)
  • Curățarea clipboard-ului micșorarea timpului de pornire a sistemului de operare prin dezactivarea unor aplicații care pornesc automat
  • Imbunătățirea performantelor hard disk-ului prin defragmentare
  • Repararea unor probleme ale sistemului de operare (ca de exemplu imposibilitatea descărcării actualizărilor, afișarea incorectă a iconițelor și a directoarelor)





ErrorViewer

Aplicație folosită pentru a depista erorile din sistem. Aplicația rulează pe sistemele de operare Windows 7 și Windows XP și are o dimensiune extrem de mică (1.58 MB).

Aplicația a fost scrisă în C# (Visual Studio 2008/NET Framework 2.0), folosind librării Open Source și editând resurse oferite gratuit.
